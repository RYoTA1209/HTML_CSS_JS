基本的な文字と字体の装飾

Color

colorプロパティは文字色の色をつけることができます。通常は文字の色ですが、text-decorationによる下線や上線などにも適用されます。 


section p{
    color:red;
}

Font families

font-familyプロパティはテキストの字体を設定することができます。閲覧しているPCにそのフォントがあればプロパティは適用されますが、ない場合ブラウザのデフォルトのフォントが使われます。 


section p{
    font-family: serif;
}

複数のフォントを指定する

複数のフォントを指定するときフォント同士の間を,で区切ります。フォントを複数指定する理由は、OSによってインストールされているOSが異なるためです。 


section p{
    font-family: "MS PGothic", "Osaka", Arial, sans-serif;
}

Font size

font-sizeプロパティの前に、文字サイズの単位について学んでいきます。

  • px 絶対的な単位
  • em 親要素に対して相対的な単位
  • rem root(html)のフォントサイズに対する相対的な単位

font-style

font-styleプロパティはフォントをイタリック調や斜体にするものです。日本語ではイタリック調と斜体は適用できないフォントがほとんどです。 


footer {
    font-style:italic;
}

font-weight

font-weightにより、文字の太さを指定できます。このプロパティは多くの値があります(-light,-normal･･･,lighter,bolder,100-900) 


section p {
    font-weight:600;
}

text-transform

text-transformは日本語には関係ありません。英字などを大文字・小文字・全角文字への変換を指定できます。 


footer {
    text-transform: lowercase;
}

text-decoration

text-transformは下線、上線、取り消し線を指定できます。 


section p {
    text-decoration:line-through;
}

Text alignment

text-alignプロパティは文字の位置を指定できます。値はleft,right,cnter,justifyの4つです。 


section {
    text-align: center;
}

Line height

line-heightプロパティは行間を指定できます。 


section p{
    line-height: 3;
}

letter spacing

letter-spacingプロパティは文字間を指定できます。 


section p{
    letter-spacing: 10px;
}

Word spacing

word-spacingプロパティは単語の間隔を指定できます。単語の間隔にはマイナスの値も指定できます。 


footer{
    word-spacing: 100px;
}

今までのをまとめて...

fontプロパティでいままでの{font-style,font-variant,font-weight},font-size,font-height,font-familyをまとめて指定できます。 


font{
    font: normal 16pt serif;
}

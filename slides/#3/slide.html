CSSとは？

CSS3とは?

CSS（Cascading Style Sheets）は、今まで組んでいたHTMLで記述されたWebページのレイアウトを指定するための言語です。

CSSの構文

CSSの構文を学ぶ前にCLASS属性とID属性の話をします

CLASS属性・ID属性

class属性はHTMLの要素に「種類名」を指定することができます。一つのHTML文書に同じクラス名を何度もつかえます。
一方、id属性はHTMLの要素に「固有の名前」を指定することができます。一つのHTML文書に同じid名は一度しか使えません。

たとえば、class属性を指定すると、そのクラス名の要素のレイアウトをまとめて指定することができます。
また、javascriptで要素を指定する際class名やid名を使って特定することができるので非常に便利です。

それではCSSの書き方を紹介します。

CSSの書き方

セレクタ{プロパティ名:値;}

css文書は上記のようにセレクタ・プロパティ名・値で書きます。
セレクタはスタイルを適用する対象(要素名,id属性,class属性)のことです。
プロパティは適用するスタイルの種類(文字色、文字の大きさ等)のことです。
一つのセレクタにプロパティ指定はいくつでもできます。

CSSを使ってみよう

まずはstyleフォルダの中に"style.css"というファイルを作ってください。

body,html要素の余白(margin)を消す

                    
                        body,html{
                            margin:0;
                            background: ＃F4F5F7;
                        }

bodyとhtml要素にはなぞに余白がつけられています。ほとんどのWebデザイナーは消しているので上記のようにセレクタは
"body,html"にしてプロパティ名は余白に関する指定を行う"margin"、値は0にしましょう。
上記の例では背景色もつけています。プロパティ名は"background"で、色の指定は値で指定します。
値はキーワード値（red,blueなど),16進数(#FFFFFFなど),RGB値,HSL値,特殊なキーワード値,グローバル値があります。

記事(article)を中央揃えに

                        
                            article{
                                width: 80%;
                                margin: 0 auto;
                            }

"width"は要素の横幅を指定できます。値を80%とすることによって親要素の横幅の80%の大きさを計算してくれます。
値の単位は"%"以外にも"px"や"em"など多数あります。
要素を中央揃えにしたいときは"margin: 0 auto;"と指定すると自動的に要素を中央にしてくれます。これは呪文みたいなものです。

画像(img)を小さく

                            
                                section img{
                                    width: 50%;
                                    display: block;
                                    margin: 0 auto;
                                }

section要素にあるimg要素のスタイルを指定するとき、セレクタは『親要素 子要素』のようにするとセレクタを限定することができます
img要素を中央にしたいのですがimgタグはインライン要素なのでデフォルトでは"margin: 0 auto;"は適用できません。
そこでimg要素を強制的にブロック要素に指定すると"margin:0 auto;"が指定できます。

セクション見出し(.section_title)の色を変えてみる

                                
                                    .section_title{
                                        color:#333333;
                                    }

セレクタにクラス名を使うときクラス名の前に".(ドット)"をつけ".クラス名"のようにします。
文字色は"color"オプションで値は先ほどの背景と同じようにできます。

ナビゲーション(nav)を整える

                                    
                                            nav{
                                                background-color: #444;
                                                padding: 3px;
                                            }
                                            
                                            nav ul{
                                                padding: 0;
                                                margin: 0;
                                            }
                                            
                                            nav ul li{
                                                display: inline;
                                                margin-right: 1em;
                                            }
                                            
                                            nav ul li a{
                                                text-decoration: none;
                                                color: white;
                                            }
                                            
                                            nav ul li a:hover{
                                                border-bottom: dotted 1px white;
                                            }
